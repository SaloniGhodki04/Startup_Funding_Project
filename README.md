# Indian Startup Funding Analysis Dashboard

## Project Description
The Indian Startup Funding Analysis Dashboard is a data analytics project developed using Python and Streamlit. The dashboard analyzes startup funding data and provides insights into funding trends, investor activities, startup performance, and industry growth through interactive visualizations.

## Features
* Overall Funding Analysis
* Startup-wise Analysis
* Investor-wise Analysis
* Month-on-Month Funding Trends
* Sector-wise Funding Distribution
* City-wise Funding Analysis
* Top Startups and Investors
* Interactive Charts and Dashboards

## Technologies Used
* Python
* Streamlit
* Pandas
* NumPy
* Plotly Express

## Data Preprocessing
* Removed unnecessary columns.
* Renamed columns for better readability.
* Cleaned date and funding amount fields.
* Standardized startup and investor names.
* Extracted year and month for trend analysis.

## Dashboard Modules

### Overall Analysis
Provides key funding metrics, sector distribution, funding trends, city-wise funding, top startups, and top investors.

### Startup Analysis
Displays startup details, funding history, total funding raised, and similar startups in the same industry.

### Investor Analysis
Shows recent investments, biggest investments, preferred sectors, funding stages, city distribution, and year-wise investment trends.

## Key Skills Demonstrated
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Dashboard Development
* Business Insights Generation

## Conclusion
This project helps users explore and understand the Indian startup ecosystem by transforming raw funding data into meaningful insights using interactive visualizations and analytical dashboards.
